overview

The old host surface was a large mixed implementation. The new shape keeps index.ts as the wiring layer and moves concrete behavior into focused modules.

  • Inbound channels normalize messages and callbacks.
  • The host chooses a run route and delegates.
  • Pipeline dispatch owns chat, coder, subagent, cron, and scheduled run flow.
  • Agent runtime streams progress through adapters and consolidated host events.
  • Persistence, memory, skills, and service loops sit behind clear boundaries.

# Ground Truth Document

## Original User Request
Farm Friend FFT Nano e-commerce marketplace for agricultural AI solutions. Site showcases hardware kits, AI skills, services, support plans, cloud services, education, consulting, and data products for modern farmers using FFT_nano edge-native AI technology.

## Site Purpose
E-commerce marketplace to sell FarmFriend products and services to agricultural customers. Demonstrates FFT_nano edge-native AI capabilities and allows customers to purchase hardware kits, AI skills, setup services, support plans, cloud services, education, consulting, and data products.

## Target Audience
- Farmers (small, medium, large operations)
- Agricultural consultants
- Technology buyers in agriculture sector
- Farm management decision makers
- Age range: 30-65, primarily professional/technical

## Required Features
- Product catalog with 8 categories (Hardware, Skills, Services, Support, Cloud, Education, Consulting, Data)
- Shopping cart system with add/remove/update quantity functionality
- Checkout integration (PayPal, Stripe)
- Newsletter signup form
- Navigation to all product sections
- Mobile-responsive design
- Product modals with detailed information
- Search functionality for products

## Branding & Tone
- Visual style: Earth-toned, professional agriculture tech aesthetic
- Color scheme: Earth dark (#1a1814), Earth mid (#2d2820), Earth light (#3d3630), Brand primary green (#4a7c59), Brand accent gold (#c4a35a), Cream (#f5f2ed)
- Typography: Space Grotesk (body), Cormorant Garamond (display)
- Tone: Professional, technical, trustworthy, agriculture-focused, modern

## Content Requirements
- Product information must be accurate for agriculture sector
- Technical specifications clearly presented
- Feature tags for quick scanning
- Pricing transparent with period indicators (monthly vs one-time)
- FFT_nano technology benefits clearly explained
- Privacy and security messaging for edge-native AI

## Technical Requirements
- Responsive: Yes (mobile, tablet, desktop)
- Forms: Newsletter signup, search, cart interactions
- Interactivity: Smooth scrolling navigation, cart sidebar, product modals, search filtering, notifications
- Performance: Fast load times (<3 seconds), minimal console errors

## Success Criteria
What makes this site "production-ready"?
- [ ] All navigation links work correctly (scroll to sections)
- [ ] Shopping cart fully functional (add, remove, update quantity, total calculation)
- [ ] Newsletter form accepts and validates email submissions
- [ ] All product "Add to Cart" buttons trigger cart actions
- [ ] Product modals display complete information
- [ ] Mobile navigation menu works (hamburger menu on small screens)
- [ ] Footer present with all required links
- [ ] Search filters products correctly
- [ ] No console errors in browser
- [ ] Checkout links configured with actual payment URLs
- [ ] Forms validate user input appropriately
- [ ] All images load correctly
- [ ] Site is fully responsive on all screen sizes

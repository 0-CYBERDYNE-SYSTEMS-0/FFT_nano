The Agent Advantage

FFT_nano is a living instruction manual. Once your agent has:

  • A primary LLM provider (OpenAI, Anthropic, Ollama, etc.)
  • Web access (via curl, browser automation)

It can sign up for services, configure integrations, and build on top of FFT_nano by itself. You're the farmer. Your agent is the farmhand.

The FFT_nano Philosophy
Most setup tasks your agent can handle. You give it a goal, it figures out the how.
Important Things to Know
  • Your agent is helpful, not infallible
  • API keys cost money
  • The agent can only work with what it can see
  • Scheduled tasks run automatically

Quickstart

1. Clone and Bootstrap

Terminal
git clone https://github.com/0-CYBERDYNE-SYSTEMS-0/FFT_nano.git
cd FFT_nano
./scripts/setup.sh

2. Configure Credentials

Edit .env with your provider API keys. See Credentials.

3. Authenticate

WhatsApp (if using)
npm run auth

4. Start

Development
./scripts/start.sh dev

Architecture

FFT_nano is a single Node.js process that receives messages from Telegram/WhatsApp, stores data in SQLite, runs the agent in a container, and sends responses back.

ComponentPurpose
src/index.tsMain router, message handling
src/whatsapp/WhatsApp Baileys integration
src/telegram/Telegram Bot API
src/scheduler/Cron-style task scheduling
container/Container build definitions
skills/Agent skill definitions
For AI Agents

The codebase is ~22K lines. Your agent can traverse the source, understand the architecture, and make informed changes.

Privacy Note
Conversations stay local. Nothing goes to external servers except via your LLM provider.

Scheduling

Schedule recurring or one-time tasks.

  • Cron: 0 6 * * * (daily at 6am)
  • Intervals: every 30 minutes
  • One-time: ISO timestamp
Scheduling Awareness
Each task uses API calls. Start with less frequent schedules.

Container Isolation

Agents run inside Apple Container (macOS) or Docker (Linux). Each invocation spawns a container with mounted directories.

What This Means
Your agent runs in an isolated sandbox. It can only see files you explicitly give it access to.
Farmer Note
Let your agent work autonomously without worrying it will mess up your system.

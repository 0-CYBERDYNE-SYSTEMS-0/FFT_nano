Container Runtime

Canonical source: docs-site/developer/03-container-runtime.md.

Key facts:

  • Runtime options: docker and host (no apple runtime).
  • CONTAINER_RUNTIME=host requires FFT_NANO_ALLOW_HOST_RUNTIME=1.
  • Main project mount to /workspace/project is read-only.
  • Runtime timeout floor is multi-hour (default 6h baseline with idle guard floor).

Security Model

Canonical source: docs-site/developer/11-security-model.md.

  • Default runtime is Docker; host runtime is optional and explicitly gated.
  • Security controls are host-enforced (mount policy, env allowlist, IPC authorization, admin gates).
  • External mount allowlist: ~/.config/fft_nano/mount-allowlist.json.

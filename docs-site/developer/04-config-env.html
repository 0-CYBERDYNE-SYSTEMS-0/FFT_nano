Configuration and Environment Variables

Canonical source: docs-site/developer/04-config-env.md.

  • Runtime selector: CONTAINER_RUNTIME=auto|docker|host (no apple runtime).
  • Assistant default is profile-dependent: core=OpenClaw, farm=FarmFriend.
  • Host runtime requires explicit allow env flags.
  • Runtime timeout defaults are multi-hour (CONTAINER_TIMEOUT/IDLE_TIMEOUT baseline 6h).
